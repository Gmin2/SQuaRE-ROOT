# Magic state production

YAML profiles for distillation, cultivation, or factory throughput models. Each file must follow `../Schemas.yaml` (`document_header` + `parameter_entry` per assumption).

Place one logical profile per file (e.g. `ccz_factory_gidney_ekera_2021.yaml`).
